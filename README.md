# 高一十班作业查看器

## 项目简介

这是一个专为高一十班设计的作业查看与管理系统，使用Python的Pygame库构建图形界面，结合Tkinter实现文本编辑功能。程序具有美观的界面、流畅的动画效果和实用的作业管理功能。

## 主要功能

### 1. 主界面展示
- 显示9个科目的作业网格（语文、数学、英语、物理、化学、生物、政治、历史、地理）
- 实时显示当前日期和星期信息
- 响应式布局设计

### 2. 作业编辑功能
- **编辑模式**：进入编辑模式后，点击任意科目框可弹出文本编辑器
- **自动保存**：关闭编辑窗口后自动保存作业内容
- **窗口管理**：编辑时点击主界面会自动将编辑窗口置顶

### 3. 历史记录查看
- **日期选择**：使用系统对话框选择年份、月份、日期
- **层级结构**：年份→月份→日期三层选择模式
- **自动筛选**：只显示有历史记录的日期

### 4. 轮播模式
- **自动轮播**：每隔10秒自动切换到下一个有作业的科目
- **手动控制**：点击左右箭头手动切换
- **全屏展示**：大字体、居中显示，便于查看

### 5. 关闭界面
- **退出**：点击关闭按钮进入专属关闭界面
- **滑块**：滑块和滑动条一起从底部缓动出现

## 安装与运行

### 快速开始
1. 建议安装Python 3.13.8
2. 安装所需依赖：
   ```bash
   pip install pygame
   ```
3. 将图片资源放入`Slide`文件夹（与主程序同一目录）
4. 运行程序：
   ```bash
   python Homework_TST.py
   ```

### 数据存储
作业数据自动保存在：
```
C:/Users/[用户名]/Documents/Homework/YYYY/MM/DD.json
```
- `YYYY`：年份（如2024）
- `MM`：月份（如01）
- `DD`：日期（如15）

程序会自动创建必要的目录结构。

## 使用指南

### 基本操作
1. **查看作业**：程序启动后默认显示今日作业
2. **进入编辑模式**：点击右上角"编辑"按钮
3. **编辑作业**：进入编辑模式后，点击任意科目框
4. **查看历史**：点击"记录"按钮，选择日期
5. **轮播模式**：点击"轮换"按钮
6. **关闭程序**：点击"关闭"按钮
